# Thumbify

Product Image Thumbfier

Demo is HERE


## Quick start

### Install

npm will be added after version 1.0.0

### Load

```html
<link rel="stylesheet" href="thumbify.css" />
```

Put the script at the bottom of your markup right after jQuery:

```html
<script src="thumbify.js"></script>
```
